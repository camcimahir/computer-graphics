# Computer Graphics Portfolio

A collection of interactive 3D graphics projects built from scratch during CSE 160 (Computer Graphics) at UC Santa Cruz. Each project progressively builds on the last — from basic 2D vector math up to a fully animated Three.js racing scene with shadow mapping and particle systems.

**[Live Site →](https://camcimahir.github.io/CSE-160/)**

## Featured Projects

### Sunset Speedway · Three.js

Animated 3D racing environment with spline-based car animation, shadow mapping, rain particles, and GLTF model loading.

## All Projects

| Project | Description | Tech |
|---|---|---|
| [Sunset Speedway](sunset-speedway/) | Animated 3D racing environment with spline-based car animation, shadow mapping, rain particles, and GLTF model loading. | Three.js |
| [Phong Lighting Engine](lighting-engine/) | Custom WebGL engine implementing Phong illumination with dynamic point lights, spotlights, normal matrices, and OBJ parsing. | WebGL, GLSL |
| [Voxel World](voxel-world/) | First-person Minecraft-inspired world with texture mapping, collision physics, gravity, and block placement. | WebGL |
| [Blocky Animal](blocky-animal/) | Hierarchical 3D modeling with matrix stacks, articulated joints, and procedural animation. | WebGL |
| [Interactive Shape Canvas](shape-canvas/) | WebGL shape renderer with OOP classes, dynamic color/size controls, and a hand-drawn triangle character. | WebGL |
| [Vector Math Visualizer](vector-visualizer/) | Canvas sandbox for testing 2D vector operations — addition, normalization, cross products, and area calculations. | Canvas API |

## Tools & Technologies

- **Languages:** JavaScript, HTML5, CSS3, GLSL
- **Graphics APIs:** WebGL, Three.js, HTML5 Canvas
- **Techniques:** Phong shading, texture/UV mapping, matrix stacks, spline animation, shadow mapping, particle systems, OBJ/GLTF loading
